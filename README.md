# OpenCV-KNN-OCR
【算法】
采用“K维最邻近分类算法”

【程序中分类器的指标】
分类：2（0表示横向，1表示竖向）
维度：10
维度和分类均可以自行调整

【说明】
目前只要求区分横向和竖向，所以分为两类。如果以后要判断角度，可以分更多类
维度和训练样本直接相关，维度越高，训练样本越多，判断就会越准确

【目录】
include：	OpenCV2.3.1头文件
lib：		OpenCV2.3.1导入库（32bit）
train_pic：	训练样本集(图片尺寸200x200)
test_pic：	测试样本集(图片尺寸200x200)

【函数】
threshold：	二值化（预处理）
getData：	取训练数据
train：		训练
classify：	分类
test：		测试
